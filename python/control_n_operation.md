# 제어와 연산

### 1. 기본 연산

Python 에서 연산은 다른 프로그래밍 언어들과 마찬가지로 간결한 기호들을 사용합니다. 아래 표에 나와있는 연산들을 사용해보며 가볍게 시작해봅니다.

| 연산 기호 | 의미 |
| :---: | :--- |
| +  -  /  \* | 사칙 연산 |
| =  !=  &lt;=  &gt;=  &lt;  &gt; | 비교 연산 |
| and  or  not | 논리 연산 |

사칙 연산은 설명이 필요없을 정도로 간단합니다. 초등학생 이상이라면 누구나 배웠을 기호를 그대로 사용하기 때문이죠.

```python
a = 1+1
print("Add :", a)

b = 1-1
print("Subtract :", b)

c = 1/2
print("Divide :", a)

d = 2*2
print("Multiply :", a)
```

```text

Subtract : 0
Divide : 0.5
Multiply : 4
```
